create a web server using bun and a frontend app using uppy framework. create a folder called uploads on the server.

1. Takes a file or multiple such files as input from the user.
2. Uploads the file or files to the uploads folder.
3. Does not allow a single file to exceed 10 MB in size.
4. Runs a scan after the upload is complete and deletes in case of virus.
5. Only allows jpg,png, doc, docx,rtf and pdf file types.
6. Shows a dynamic animation to the user on the progress of the upload.
7. Prohibits the bulk uploads size to exceed 500 MB.
8. The name of the file should be retained.
9. Do not mention the name of the scanner.
